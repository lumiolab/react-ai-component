AI React Component ​

Generate React components from prompts using AI.

Introduction ​

ai-react-component lets you write:

js
const Button = ai`clean modern button`

…and get a working React component.

It works by:

  1. Sending your prompt to an AI
  2. Generating JSX code
  3. Compiling it in the browser
  4. Executing it as a React component

Installation ​

bash
npm install ai-react-component
bash
yarn add ai-react-component
bash
pnpm add ai-react-component
bash
bun add ai-react-component

Peer dependency:

  • react

How It Works ​

  1. Prompt: ai\clean modern button``
  2. AI Generation: Calls OpenAI with system instructions
  3. Validation: Checks for export default and blocks imports
  4. Compilation: Babel compiles JSX → JS
  5. Execution: Runs code via new Function
  6. Rendering: React renders the component
  7. Caching: Result stored in memory
